## 克隆代码到本地

```shell

# github
git clone https://github.com/htmlxudong/vue3-tiptap.git

# gitee
git clone https://gitee.com/xd_web/vue3-tiptap.git

```

## 安装依赖 📦

```
pnpm install or npm install

```

## 运行项目 🚀

```
npm run dev

```

## 技术栈 🥇

核心：vue3 + ant-design-vue + vite

代码提交：husky、commitlint

代码格式化：preitter
